# MR Cocktail
## About
Mr Cocktail is a website for cocktail lovers to uplaod their favorit cocktail recipe <br>
Users can choose ingredients from a list and upload a photo <br>
Using:
* Ruby on Rails
* PostgreSQL
* Javascript ES6
* Bootstrap
* SCSS
* HTML5
## Project Feature
* Client side photo upload
* CRUD operations
